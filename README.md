# <h1>🚀 Complete React Learning Repository</h1>

A comprehensive React.js learning repository covering React fundamentals, hooks, routing, forms, custom hooks, state management, data fetching, and testing with practical examples.

This repository is designed for beginners, students, and interview preparation. Every topic includes easy-to-understand code examples to help build a strong React foundation.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/complete-react-learning.git
```

Go to project directory

```bash
cd complete-react-learning
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
